Como surgiu a Internet

  • A principio as comunicações mais praticas eram realizadas atraves de Telégrafos utilizando os códigos morses. Em 1958 a primeira comunicação intercontinental foi realizada quando atravessaram um cabo de um continente ao outro.
  • Na década de 1950 devido a guerra fria, as tecnologias explodiram em relevancia no mundo devido a necessidade gigantesca de realizar calculos extremos para adiantarem no processo da corrida até a lua.
  • Nesse meio surgiu a ARPA uma agencia de projetos de pesquisas que acabou por se tornar a DARPA.
  • Nessa epoca um homem chamado **Joseph licklider**, deu a primeira ignição para a construção da internet, teorizando sobre uma rede onde todos poderiam se comunicar atraves dela.
  • Passado algum tempo surgiu a ARPANET que era um sistema de comunicação de pacotes, enviados um a um e que permitia multiplos destinos, a primeira conexão entre dois destinos foi realizada em 2910/1969, do qual a palavra "LOGIN" foi enviada, contudo apenas "LO" chegou, sendo assim também a primeira queda de conexão da historia.
  • Vinton Cerf e Robert Kahn criaram o TCP/IP e implantaram isso na ARPANET o que fez com que essa rede criasse mais robustes e com isso fossem sendo criadas microredes.
  • Em 1989, Tim Berners-Lee viu que alguma hora devido a criaçõa de diversas microredes por diversos fornecedores tudo iria virar uma bagunça e entã uniu o hipertexto com o TCP/IP e criar a World Wide Web(WWW) , o primeiro navegador.

Gerações dos computadores

O conceito de computador é algo já antigo, desde Alan turing, contudo o primeiro computador que consideramos é o Mark I( 1944) um computador eletro-mecanico feito para a realização de calculos.

  1. A primeira geração de computadores é marcada a partir de 1946 do qual o Colossus foi criado pela alemanha tinha como principio calculos relacionados a guerra, assim como o ENIAC(1946) que tinha o mesmo principio mas sendo dos EUA, contudo o ENIAC já era um computador automatico.
  2. A segunda geração de computadores(1959 a 1965) é marcada pelos transistores permitindo assim diminuir seus tamanhos drasticamentes
  3. A terceira geraçãoo(1965 a 1970) é marcada pelos circuitos integrados, ou seja, os chips, que fizeram com que os computadores diminuissem ainda mais seus tamanhos.
  4. A quarta geração(a partir de 1971) é muito marcada pelos PC(Personal Computer) foi nessa geração também que a apple desenvolveu o primeiro mouse.
  5. A quinta geração é marcada por discussões e polemicas ainda sobre se já estamos nela ou não, devido aos computadores quanticos, IAs, Metaverso, etc.

O que é HTML

HTML ou de forma extensa HyperText Markup Language(Linguagem de marcação de HiperTexto) é uma das linguagens de marcação mais conhecidas, que utiliza um formato simples de código, podendo ser gerado sem a ajuda de aplicativos especiais Para criar um documento HTML, tudo que é necessário é de um editor de texto ASCII tal como o bloco de notas. Caso haja a necessidade de verificar a formatação do documento durante a criação, voce pode usar qualquer navegador, pois todos eles são capazes de interpretar o código. O HTML foi inventado em 1990 por um cientista chamado Tim Berners-Lee. Seu proposito inicial era de tornar possivel o acesso e a troca de informações e de documentação de pesquisas, entre cientistas de diferentes universidades. O projeto inicial se tornou um sucesso e com isso foi crescendo até se tornar o que é hoje.

O HTML tem uma função importante em aplicações web, que são programas que você incorpora em sites para gerenciar tarefas adicionar como por ex. o gerenciamento de pedidos de usuários. As aplicações web aumentam a interação do usuário na web. Quando você entra com alguma informação em uma página web, o navegador a envia para um conjunto especial de programas chamados Common Gateway Interface(CGI) scripts. Estes programas processam a informação e exibem o resultado no navegador segundos após o requerimento.

O que são TAGS?

As chamadas tags são elementos básicos de um documento HTML usados para especificar o formato do conteúdo de uma página web.

Os sinais <> permitem aos navegadores distinguir as tags de texto comum e o conteúdo delas pode ser digitado tanto em letras maiúsculas, quanto em minúsculas.

Um documento HTML possui duas partes:

  • Head(cabeçalho)
  • Body(Corpo)

No Head, voce entra com o titulo da página web, que será exibido apenas na barra de endereços do navegador.

No Body, voce entra com o conteúdo da página web

Protolos da internet

Os Protocolos consistem em um conjunto de regras para a comunicação em rede. Eles determinam como computadores se comunicam: como iniciar, como manter e como encerrar a comunicação.

  • HTTP – Hyper Text Transfer Protocol é o protocolo usado pelos servidores web para transmitir documentos HTML pela internet.
  • FTP – File Transfer Protocol é outro protocolo muito utilizado para transmitir arquivos, porém sua inferface gráfica não é tão agradável como o HTTP.
  • TCP – Transfer Control Protocol é um conjunto de protocolos para a transmissão de informação pela web, com função de verificar se os dados são transferidos de forma correta, na sequencia apropriada e sem erros.
  • IP – Internet Protocol é um protocolo de endereçamento, que fornece o endereço dos computadores da rede.
